myFlix

This application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up update their personal information and create a list of their favorite movies.

Request URL HTTP Method Response Data
Body Format
Get a list of movies /movies READ A JSON object holding data about all movies
Return data about a single
movie by title to the user		 /movies/[title] READ A JSON object holding data(title, genre, director) about a single movie by title to the user
Return data about a
genre(description) by name/title (e.g., "Thriller") 		/movies/genre/[genre] READ A JSON object containing movie titles and other data by genre
Return data about a director by name /movies/director/[directorName] READ A JSON object showing data about the director
Allow new users to register /user POST N/A
Allow users to update their user info(username) /user/[username] PUT N/A
Allow users to add a movie to their list of favorites /user/[username]/movies/[movieID] PUT A text to the user showing that a movie has been added
Allow users to remove a movie from their list of favorites /register/[username]/movies/[movieID] DELETE A text to the user showing that a movie has been removed
Allow existing users to deregister /user/[username] DELETE A text to the user showing that their email has been removed
